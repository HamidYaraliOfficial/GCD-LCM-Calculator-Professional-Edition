# GCD & LCM Calculator – Professional Edition

## English

### Overview
**GCD & LCM Calculator – Professional Edition** is a sleek, modern desktop application for computing **Greatest Common Divisor (GCD)** and **Least Common Multiple (LCM)** of two positive integers. Built with **Python** and **PyQt6**, it features a **gradient UI**, **animated sidebar**, **real-time calculations**, **multilingual support**, **dynamic theming**, and **RTL layout** — delivering a premium mathematical tool for students, educators, and professionals.

### Key Features
- **Instant GCD & LCM**: Real-time results with zero delay.
- **Input Validation**: Accepts only positive integers.
- **Dual Result Display**: Clean, large, and color-coded output.
- **Multilingual Interface**: Full support for **English**, **فارسی (RTL)**, **中文**, and **Русский**.
- **5 Stunning Themes**:
  - Light
  - Dark
  - Windows Default
  - Red
  - Blue
- **RTL Layout Support**: Automatic right-to-left for Persian.
- **Modern Sidebar Navigation**: Smooth page switching.
- **Professional Typography**: Segoe UI with bold weights.
- **Gradient Backgrounds**: Subtle, elegant depth.
- **Form Alignment**: Precise, balanced layout.

### Requirements
- Python 3.8+
- PyQt6

### Installation
1. Ensure Python is installed.
2. Install dependency:
   ```bash
   pip install PyQt6
   ```
3. Run the app:
   ```bash
   python gcd_lcm_calculator.py
   ```

### Usage
- **Enter Numbers**: Type two positive integers.
- **Click Calculate**: Choose **GCD** or **LCM**.
- **View Results**: Instantly displayed in large, bold format.
- **Switch Tabs**: Use sidebar to access **Settings**.
- **Change Language/Theme**: Use dropdowns and radio buttons.

## 中文

### 概述
**最大公约数与最小公倍数计算器 – 专业版** 是一款优雅现代的桌面应用程序，用于计算两个正整数的**最大公约数 (GCD)** 和 **最小公倍数 (LCM)**。使用 **Python** 和 **PyQt6** 构建，具备**渐变界面**、**动态侧边栏**、**实时计算**、**多语言支持**、**动态主题**和 **RTL 布局**，为学生、教师和专业人士提供高端数学工具。

### 主要功能
- **即时 GCD & LCM**：零延迟结果。
- **输入验证**：仅接受正整数。
- **双结果显示**：清晰、大号、彩色输出。
- **多语言界面**：完全支持 **英语**、**波斯语（RTL）**、**中文** 和 **俄语**。
- **5 种惊艳主题**：
  - 明亮
  - 暗黑
  - Windows 默认
  - 红色
  - 蓝色
- **RTL 布局支持**：波斯语自动右到左。
- **现代侧边栏导航**：流畅页面切换。
- **专业排版**：Segoe UI 加粗字体。
- **渐变背景**：细腻视觉深度。
- **表单对齐**：精确平衡布局。

### 要求
- Python 3.8+
- PyQt6


### 使用方法
- **输入数字**：键入两个正整数。
- **点击计算**：选择 **GCD** 或 **LCM**。
- **查看结果**：立即以大号粗体显示。
- **切换标签**：使用侧边栏访问 **设置**。
- **更改语言/主题**：使用下拉菜单和单选按钮。
